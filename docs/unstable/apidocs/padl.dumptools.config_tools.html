padl.dumptools.config_tools

padl.dumptools.config_tools.apply_params(source, parsed_kwargs, scope)

Apply param replacement in source.

Parameters

  • source – The sourcecode in which to replace the params.

  • parsed_kwargs – The parsed keyword argument with params to replace.

  • scope – The scope.

padl.dumptools.config_tools.change_param(source: str, param_name: str, value: str) → str

Change the value of a param with name param_name in source to value.

Note that value is the string representation of the value.

Example:

>>> source = "x = param('myparam', 1)"
>>> change_param(source, "myparam", "100")
"x = param('myparam', 100)"
padl.dumptools.config_tools.change_params(source, param_name, **kwargs)

Change the value of params with name param_name in source to values provided as kwargs.

Note that values must be the string representation of the values.

Example:

>>> source = "x = params('myparam', a=1, b=2)"
>>> change_params(source, "myparam", a="100")
"x = params('myparam', a=100, b=2)"
padl.dumptools.config_tools.extract_param_s(source: str) → dict

Extract params() calls from source.

Example:

>>> source = '''a = param('param_a', x, 'a parameter')
... b = param('b', 2)'''
>>> extract_param_s(source)  
{'param_a': {'name': "'param_a'", 'val': 'x', 'description': "'a parameter'",
             'use_default': 'True',
             'position': Position(lineno=1, end_lineno=1, col_offset=21, end_col_offset=22)},
 'b': {'name': "'b'", 'val': '2', 'description': 'None', 'use_default': 'True',
       'position': Position(lineno=2, end_lineno=2, col_offset=15, end_col_offset=16)}}
Parameters

source – The source code from which to extract the param()-calls.

Returns

A dict mapping param() arguments (name, val, description, use_default) to their string values. Also adds a “position” entry with the position of the parameter’s value in the code.

padl.dumptools.config_tools.extract_params_s(source)

Extract params() calls from source.

Example:

>>> source = '''x = X(**params('params_x', a=1, b=2))'''
>>> extract_params_s(source)  
{'params_x':
    {'name': "'params_x'",
     'kwargs': {'a': '1', 'b': '2'},
     'use_defaults': 'True',
     'allow_free': 'False',
     'positions': {'a': Position(lineno=1, end_lineno=1, col_offset=29, end_col_offset=30),
                   'b': Position(lineno=1, end_lineno=1, col_offset=34, end_col_offset=35)},
     'end_position': Position(lineno=1, end_lineno=1, col_offset=35, end_col_offset=35)}}
Parameters

source – The source code from which to extract the params()-calls.

Returns

A dict mapping params() arguments (name, use_defaults and allow_free) to their string values. Also adds a “kwargs” entry and a “positions” entry with the position of the parameter’s values in the code.

padl.dumptools.config_tools.param(name, val, description=None, use_default=True)

Helper function for marking parameters.

Parameters can be overridden when loading. See also padl.load().

Parameters

  • name – The name of the parameter.

  • val – The default value of the parameter / the value before saving.

  • description – Description of the parameter.

  • use_default – If True, will use val when loading without specifying a different value.

Returns

val

padl.dumptools.config_tools.params(name, *, use_defaults=True, allow_free=False, **kwargs)

Helper function for marking parameters.

Parameters can be overridden when loading. See also padl.load().

Parameters

  • name – The name of the parameter.

  • description – Description of the parameter.

  • use_defaults – If True, use passed keyword args when loading without specifying different values.

padl.dumptools.config_tools.parse_dict(source)

Parse a string containing a python dict.

Example:

>>> parse_dict("{'a': 1, 'b': '2', 'c': random.rand(1000)}")
{'a': '1', 'b': "'2'", 'c': 'random.rand(1000)'}
Parameters

source – A string containing a python dict.

Returns

The dict. Note that the values are strings of the values.
